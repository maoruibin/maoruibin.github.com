大家好，我是咕咚。

这周事情特别多，自己一直处在一种兴奋、有时又焦虑的状态，起起伏伏。

简单记录一下这周都做了哪些事。

## 1. SlideNote：从想法到发布，两天

昨天发布了 SlideNote，一个 Chrome 浏览器侧边栏笔记插件。

其实这是周三早晨突然冒出来的想法。之前一直在用老插件，越用越不舒服，突然想到 Chrome 有 Side Panel API，再加上这段时间一直在用 Claude Code，就觉着——有想法的话，实现其实特别简单。

早晨建了个文件夹，把想法告诉 AI，简单聊了一些产品想法，很快他就帮我实现了。

整个项目一行代码没写，除了点子是我的，开发、文章、封面图、推广文案，全程都是 AI 帮着做的。完全是一个 AI 驱动的项目，我则是一个 AI 代理人。

[文章在这里：侧边笔记，常伴左右：我用 Claude Code 开发了一个浏览器插件](https://mp.weixin.qq.com/s/nivLWAkYWiMHRHMENko0UQ)

## 2. 被阮一峰网络日志收录

[科技爱好者周刊#381：中国 AI 大模型领导者在想什么](https://mp.weixin.qq.com/s/1pfZZvpb12-P3wvdPuYfxQ?scene=1&click_id=36)

SlideNote 的代码被我开源了，我顺便就把项目提交到了阮一峰的网络日志，他每周五会发布互联网上收集到的各种工具和信息。

之前做的一些工具也提交过，但一直没被收录。这次很幸运，被收录了。

这个公众号每周五更新，内容很丰富，也欢迎大家关注。

## 3. AI Coding 的一周

这周是正式使用 Claude Code 的一周。

之前也在用，但越用越熟练，对他的认识越来越多。

AI 的发展确实特别快，业余项目现在都是用 AI 去开发了。

## 4. Skills 学习

Skills 是 ClaudeCode 发布的一个新标准，照猫画虎，跟着几个项目参考了一下，然后就上手了，其实挺简单的，就是一个思路转变而已。

这周开发了好几个技能，很多没有公布，自己用就够了。

技能搭配 Claude Code 会特别好用。比如图床上传技能——本地图片，一句话直接上传到图床。写文章的时候特别有用。

之前需要找个工具，上传，现在只需要一句话。技能这个东西，简直是大发明。

## 5. 公众号数据最好的一周

这周公众号的数据，是我做公众号以来最好的一周。好几篇文章都好几千的阅读量。

其中一篇文章《[X：我今年最好的信息源](https://mp.weixin.qq.com/s/3jqJmbBEeEBSx3ss3MxrFFg)》，现在已经 8000 多阅读量了。这是我写公众号这么长时间，阅读量最高的一篇文章。

发布前其实犹豫过，不一定有人关心。结果发布后，在微信生态里一下子被引爆了。

看来在微信生态里，大家对于“打破信息差”的需求是被压抑的

但我不会因为一篇文章阅读量高，就围绕那个主题去写。还是会保持自己的节奏。

## 7 AI 的割裂感

业余项目，全程用 AI 开发，效率特别高，而且不论是技术项目还是非技术项目，完全都用它去做。

让它做设计稿，比如 SlideNote 的封面图、inBox 笔记更新日志的封面图，AI 都设计得特别好看。

跟它说要求，他真的能帮你做到。

SlideNote 那个封面图，一开始设计得不好，我说"帮我对齐一下"，效果立马立竿见影。

![](https://gudong.s3.bitiful.net/weimd/1768443166661_image.png)


那种感觉，就是它懂你。

上架市场的时候，市场要求截图的尺寸，我把要求粘给 AI，让他做图。很快就做了一个设计，做成网页，一截图——刚好就是那个大小。真好。

跟 AI 合作，就像跟一个非常优秀的人合作。有诉求，他有求必应。体验很愉悦。

但是——公司的项目，用 AI 很难进行。[今天 code review 了两小时](https://mp.weixin.qq.com/s/eCOIDtlBsQczVK0zx1vaJg)

也尝试了，但企业项目尤其我所在的公司代码量特别臃肿，AI 不能很好地找到问题所在。这周用它解决一个 UI bug，整整解了两天，各种想法，最后 AI 还是没能完全解了。

这就是现在的现状：

在个人项目里，AI 是一把激光剑；在公司项目里，它可能只是一把指甲刀。

这种落差感，让人挺无奈的。

## 8. 休息与放松

工作越兴奋，身体越容易透支。

虽然在写代码时多巴胺分泌让你感觉不到累，但一旦松弛下来，疲惫感会像潮水一样涌上来。

所以我强制自己休息。中午吃完饭必须去户外溜达，看看树，看看阳光。

这周打了两次篮球。现在打球不是为了输赢，就是为了出汗，为了让眼睛离开屏幕。

不要受伤，开心就好。

## 写在最后

这周是“输入与输出”都极高的一周。

AI 让创造变得容易上瘾，但我们毕竟是肉体凡胎。

享受多巴胺的同时，别忘了去楼下走走，别忘了你的篮球和阳光。

我是咕咚，祝大家周末愉快。
